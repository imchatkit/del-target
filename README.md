## 🎯 产品概述

**SystemOptimizer（系统优化大师）** 是一款集文件夹清理和进程管理于一体的Windows桌面应用程序。通过直观的图形界面，帮助用户快速清理系统垃圾、管理运行进程，显著提升系统性能和磁盘空间利用率。

### ✨ 核心特色

- **🎯 智能模糊匹配** - 支持通配符和正则表达式，精准定位目标
- **🔍 深度递归搜索** - 多层级目录遍历，无遗漏清理
- **⚡ 批量操作** - 一键处理多个文件夹或进程，效率倍增
- **🛡️ 安全机制** - 多重确认和预览功能，避免误操作
- **📊 实时监控** - 进程CPU、内存使用率实时显示
- **💾 配置记忆** - 智能保存用户偏好设置

## 🚀 双核心功能

### 🗂️ 文件夹清理工具
**专业的磁盘空间优化解决方案**

#### 核心能力
- **智能搜索引擎**：支持多种匹配模式（`*cache*`、`node_modules`、`target`等）
- **深度扫描**：1-5层或无限制递归搜索，覆盖所有嵌套目录
- **安全预览**：删除前详细展示匹配结果，包含文件夹大小和路径
- **批量清理**：一次性删除所有匹配的文件夹
- **配置管理**：保存常用清理方案，提高工作效率

#### 典型应用
- **开发环境清理**：清理`node_modules`、`target`、`build`等构建产物
- **系统垃圾清理**：清理临时文件夹、缓存目录
- **项目维护**：批量清理多个项目的编译输出

### ⚡ 进程管理器
**高效的系统进程监控与管理工具**

#### 核心能力
- **实时监控**：显示PID、进程名、CPU使用率、内存占用、运行状态
- **智能过滤**：模糊匹配快速定位目标进程（如`*jdk*`、`chrome*`）
- **批量操作**：支持选择性关闭或全量关闭匹配进程
- **安全保护**：内置系统进程保护机制，防止误关键进程
- **操作日志**：详细记录每次操作，便于追踪和审计

#### 典型应用
- **内存优化**：关闭多余的Java、Chrome、IDE进程
- **性能调优**：清理高CPU占用的后台进程
- **开发环境管理**：快速重启开发工具和服务

## 📦 快速开始

### 💻 系统要求
- **操作系统**：Windows 10/11 (64位)
- **内存**：最小2GB RAM
- **磁盘空间**：50MB可用空间
- **权限**：建议管理员权限以获得最佳体验

### 🚀 安装使用

#### 方式1：直接运行（推荐）
```bash
# 下载并运行exe文件
.\FolderCleaner.exe
```

#### 方式2：源码运行
```bash
# 克隆仓库
git clone [repository-url]

# 安装依赖
pip install -r requirements.txt

# 运行主程序
python folder_cleaner.py
```

### 🎯 快速上手

1. **文件夹清理**
   - 选择目标目录
   - 输入匹配模式（如：`*cache*`）
   - 预览匹配结果
   - 执行清理操作

2. **进程管理**
   - 点击"进程管理器"按钮
   - 搜索目标进程（如：`*jdk*`）
   - 选择关闭方式
   - 查看操作结果
